# 📦 설치 가이드

이 문서는 주간 업무보고서 자동 생성 도구를 처음 설치하는 사용자를 위한 가이드입니다.

## 🚀 빠른 설치 (Windows)

### 1단계: 저장소 다운로드

**방법 A: ZIP 파일 다운로드**
1. GitHub 저장소 페이지에서 `Code` 버튼 클릭
2. `Download ZIP` 선택
3. 원하는 위치에 압축 해제

**방법 B: Git Clone**
```bash
git clone https://github.com/Frichel/weekly-report-generator.git
cd weekly-report-generator
```

### 2단계: 자동 설치 실행

다운로드한 폴더에서 `setup.bat` 파일을 **더블클릭**하세요.

<parameter name="waitForPreviousTools">false

설치 완료 후에는 INSTALL.md 파일은 삭제해도 좋습니다.
